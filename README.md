# Python
Учебно-методические материалы по дисциплине "Программирование на Python". 
Репозиторий включает:  
-Конспекты лекций  
-Методические указания к лабораторным работам  
-Варианты заданий  
-Дополнительные обучающие ресурсы
------------------------------------------------------------------------------------------------------------------------------------------------
Детальная система оценивания учебного курса по Python
(100-балльная система с обязательной сдачей всех лабораторных работ для допуска к экзамену)

1. Общие положения
Максимальный балл за курс: 100

Компоненты оценки:
Лабораторные работы: 60 баллов (8 работ × 7.5 баллов)
Экзамен: 40 баллов

Обязательное условие:
Для допуска к экзамену студент должен сдать все 8 лабораторных работ (минимум 3 балла за каждую).
-------------------------------------------------------------------------------------------------------------------------------------------------
2. Оценивание лабораторных работ
2.1. Балльная структура
Каждая из 8 лабораторных работ оценивается по 7.5 баллов.
Максимальный балл за работу: 7.5
Минимальный проходной балл за работу: 3

2.2. Критерии оценки
7.5	-> Отлично	Работа выполнена полностью, код работает без ошибок, оформлен по стандартам (PEP 8), соблюдены все требования задания.
6.0–7.0 ->	Хорошо	Работа сдана с незначительными недочётами (например, мелкие ошибки, неоптимальное решение, небольшие отклонения от требований).
4.0–5.0 ->	Удовлетворительно	Работа выполнена частично, есть серьёзные ошибки, но код в целом решает задачу.
0.0–3.0 ->	Неудовлетворительно	Работа не выполнена, код нерабочий или нарушены базовые требования.

2.3. Правила сдачи и пересдачи
Дедлайны:
За сдачу работы после срока максимальный балл — 5.0 (даже если работа идеальна).
Штраф за просрочку: -1 балл за каждую неделю задержки (но не ниже 3.0 баллов).

Пересдачи:
Студент имеет право на 1 пересдачу каждой работы.
Максимальный балл после пересдачи — 5.0 (даже если исправлены все ошибки).

Плагиат:
В случае обнаружения списывания работа оценивается в 0 баллов без права пересдачи.
-------------------------------------------------------------------------------------------------------------------------------------------------
3. Экзамен (40 баллов)
3.1. Условия допуска
Студент допускается к экзамену только после сдачи всех 8 лабораторных работ (минимум 3 балла за каждую).
Если хотя бы одна работа не сдана — экзамен не разрешён, курс считается незачтённым.

3.2. Критерии оценки экзамена
36–40 ->	Отлично	Полное и грамотное решение всех задач, чистый и эффективный код, глубокое понимание темы.
28–35 ->	Хорошо	Решены все основные задачи, но есть незначительные недочёты.
20–27 ->	Удовлетворительно	Решена только часть задач, есть серьёзные ошибки, но видно понимание основ.
0–19 ->	Неудовлетворительно	Критичные ошибки, нерешённые задачи, отсутствие понимания темы.
------------------------------------------------------------------------------------------------------------------------------------------------
4. Итоговая оценка
Итоговая оценка вычисляется как сумма баллов за лабораторные работы и экзамен:
85–100 ->	5 (Отлично)	Высокий уровень знаний, все работы сданы на высокие баллы.
70–84 ->	4 (Хорошо)	Стабильные результаты, возможны мелкие недочёты.
60–69 ->	3 (Удовлетворительно)	Минимальный проходной уровень.
0–59 ->	2 (Неудовлетворительно)	Курс не зачтён.
